#Arcade Frogger Game
--------------------

This project utilizes concepts of Object Oriented Javascript programming and HTML Canvas.

#Objective
----------

Objective is to avoid the bugs and cross the water to accumulate as many points as you can.

Steps:
------
Step 1: Modify and write the code to render the images, handle the keystrokes and reset the player after collision in 'app.js'

Step 2: Modify the 'index.html' to display the score on the webpage.

Opening the game:
-----------------

Open the index.html in the web browser on your system and play.

#How to Play
------------

Up Arrow    : Move Up
Down Arrow  : Move Down
Left Arrow  : Move Left
Right Arrow : Move Right
